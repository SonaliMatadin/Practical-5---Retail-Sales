# Retail-Sales
Practical: SQL Fundamentals (Snowflake – Basic SQL Syntax)

This practical focuses on learning and applying core SQL concepts in Snowflake using the retail_sales_dataset.csv. The exercise builds foundational querying skills by working with real transaction data from a retail environment.

✅ Key Concepts Covered

SELECT & SELECT DISTINCT

WHERE filtering and conditional logic

Aggregate functions (COUNT, AVG, MAX, MIN, SUM)

GROUP BY & HAVING

CASE statements for categorization

✅ Practical Outcomes

Retrieve full and partial datasets

Filter records based on numeric and category criteria

Summarize sales metrics like revenue, quantity, and averages

Group data by product category and customer gender

Create new calculated fields such as Spending Level and Age Group

✅ Dataset Highlights

Transaction details (ID, Date, Customer)

Customer demographics (Gender, Age)

Product metrics (Category, Quantity, Price, Total Amount)

Tool Used: Snowflake (data upload, querying, execution)
